# 低功耗模式

## 功能概述

v3.7.1 新增。角色长时间空闲时自动降低绘制帧率，减少 CPU/GPU 占用。

## 工作逻辑

```
idle 状态 + 30 秒无鼠标移动
    ↓
帧率降至 15fps（每 4 帧绘制 1 帧）
    ↓
鼠标移动
    ↓
立即恢复 60fps 全速绘制
```

## 技术实现

`renderer/app.js` 的动画循环中：

```javascript
// 检查空闲时间
const idleTime = now - M.lastActivity;
const isIdle = M.FSM.state === 'idle';

if (isIdle && idleTime >= 30000) {
  // 低功耗：帧跳过
  M._idleFrameCounter++;
  if (M._idleFrameCounter >= 4) {  // 60÷15=4
    M._idleFrameCounter = 0;
    draw();
  }
} else {
  // 全速：每帧绘制
  draw();
}
```

## 关键参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `_idleLowPowerThreshold` | 30000 (30s) | 进入低功耗前的空闲等待时间 |
| `_idleLowPowerFps` | 15 | 低功耗目标帧率 |
| `_idleLowPowerSkip` | 4 | 帧跳过倍数 (60÷15) |

## 唤醒条件

以下操作会重置 `M.lastActivity`，立即退出低功耗模式：

- 鼠标移动（`mousemove`）
- 体型切换
- 状态变更（idle → 其他）
- 气泡显示/更新

## 设计考量

- **FPS 选择 15 而非更低**：15fps 下呼吸动画仍然平滑，更低会出现明显卡顿
- **帧跳过而非降低 rAF 频率**：保持与浏览器刷新率同步，避免撕裂
- **仅在 idle 时生效**：working/happy 等动画状态不受影响
- **切换体型时自动重置**：`Layout.applySize()` 触发 `mousemove` 事件重置计时器
