# 文件拖放

## 功能概述

拖放文件/文件夹到桌宠窗口 → 自动递归复制到晓风数据目录 → 后端通知 → 视觉反馈。

## 交互流程

```
用户拖放文件
    ↓
dragover → 显示虚线高亮遮罩 (#overlay)
    ↓
dragleave → 隐藏遮罩
    ↓
drop → 递归复制文件/文件夹
    ↓
working 状态（张嘴吃文件动画）
    ↓
进度条气泡 (nom-progress)
    ↓
POST /api/file/process → 晓风后端
```

## 目标目录

默认目标：`D:\xiaofeng_agent\02_数据输入`

可在 `config.json` 中修改：

```json
{
  "targetDir": "../02_数据输入"
}
```

路径相对于 `拟态开发版/` 目录解析。

## 视觉反馈

| 阶段 | 效果 |
|------|------|
| 拖入 | 虚线高亮边框（白色半透明） |
| 复制中 | working 状态（张嘴 + nom 节奏 ~400ms）+ 碎屑粒子 + nom 音效 |
| 进度 | 气泡显示 `nom-progress` 进度条 |
| 完成 | 气泡显示 "(∗´▽`∗) 复制完成！" |

## 技术细节

- 使用 `webUtils.getPathForFile()` 获取文件路径（Electron 安全限制）
- 使用 Node.js `fs` 模块递归复制
- 目录处理：复制整个文件夹及其子内容
- 复制过程中窗口保持交互（不阻塞）

## 后端通知

复制完成后发送 POST 请求：

```json
POST http://127.0.0.1:5001/api/file/process
{
  "path": "D:\\xiaofeng_agent\\02_数据输入\\report.pdf",
  "filename": "report.pdf"
}
```

`apiEnabled` 设为 `false` 时跳过此步骤。
