# 桌面交互

## 透明窗口

拟态桌宠窗口是无边框、透明的 Electron BrowserWindow：

```
frame: false          → 无标题栏
transparent: true     → 背景透明
alwaysOnTop: true     → 始终在最前（screen-saver 级别）
skipTaskbar: true     → 不在任务栏显示
```

窗口尺寸随体型切换和气泡内容动态调整。

## 桌面漫游

角色可以在整个桌面范围内自由移动。整个电脑桌面都是它的领地。

- **自主闲逛**：AI 每 3 秒决策，35% 概率走向随机屏幕位置
- **好奇心**：鼠标靠近（150-500px 内）时，偶尔会走过去看看
- **Ctrl+W**：切换自主闲逛开关
- **右键菜单 → "🚶 开始闲逛"**：手动触发/停止
- **移动方式**：一步一点走过去（80px/s 慢走），不是闪现

角色通过世界坐标（`worldX`, `worldY`）定位，窗口实时跟随 `win.setPosition()`。

## 召唤系统

三种方式让角色走到你指定的位置：

| 方式 | 操作 |
|------|------|
| 右键菜单 | → "🔮 召唤到这里" → 点击桌面任意位置 → 角色跑过去 |
| 三击角色 | 快速连击角色三次 → 自动进入召唤模式 |
| 文字指令 | 按 Enter → 输入 "过来" / "召唤" / "summon" |

召唤时使用 160px/s 跑步速度，到达后显示好奇表情。

## 窗口拖拽

- 鼠标按住角色拖动窗口到新位置
- 拖拽时自动停止当前行走，世界坐标同步更新
- 拖拽结束自动保存位置到 `config/position.json`
- 通过 IPC `window-drag`（delta 模式）通知主进程

## 点击反应

| 操作 | 反应 |
|------|------|
| 点击头部 | 歪头 + 弹跳 + chirp 音效 + 心形粒子 + 泡泡"好痒~" |
| 点击身体 | 跳起 + 张大嘴 + boing 音效 + 火花粒子 + 泡泡"别戳我肚子！" |
| 双击 | 大弹跳 + 举手 + spark 粒子 + boing 音效 + 2.5s happy |
| 三击 | 自动进入召唤模式 |

## 右键菜单

| 选项 | 功能 |
|------|------|
| **体型：大/中/小** | 切换角色尺寸（110/80/50px） |
| **🔮 召唤到这里** | 进入召唤模式 |
| **🚶 开始闲逛** | 切换自主闲逛开关 |
| **穿透模式** | 勾选后鼠标事件直达下层窗口 |
| **退出** | 关闭桌宠（`Ctrl+Q` 也可） |

## 体型切换

切换时有缩放过渡动画：旧尺寸缩放值平滑过渡到 1.0，角色始终居中。

## 穿透模式

日常不遮挡操作：

| 操作 | 方式 |
|------|------|
| 开启穿透 | 托盘菜单 / 右键菜单 / `Ctrl+Shift+P` |
| 关闭穿透 | 同上 |
| 启动时默认穿透 | `config.json` → `passthrough: true` |

穿透状态下鼠标事件直达下层窗口，仅保留 `Ctrl+Shift+P` 全局快捷键和 `Ctrl+Q` 退出。

## 文字对话

按 Enter 键弹出输入框，支持：
- 自然语言聊天（Mock 关键词匹配 / 晓风后端 API）
- 指令关键词："闲逛"/"散步"（开关漫游），"过来"/"召唤"（召唤模式），"听歌"（耳机模式），"停止"/"休息"（退出特殊状态）
