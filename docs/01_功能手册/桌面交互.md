# 桌面交互

## 透明窗口

拟态桌宠窗口是无边框、半透明的 Electron BrowserWindow：

```
frame: false          → 无标题栏
transparent: true     → 背景透明
alwaysOnTop: true     → 始终在最前
skipTaskbar: true     → 不在任务栏显示
```

窗口尺寸由布局公式动态计算（参见 `00_核心概念/架构设计.md`），随体型切换和气泡内容自动调整。

## 窗口拖拽

- **实现**：Pointer Capture 拖拽（`interaction/drag.js`）
- WSL2 兼容模式下退化为 document 级 mousemove 追踪
- 拖拽结束自动保存位置到 `config/position.json`
- 移动时通过 IPC `window-drag` 通知主进程更新窗口坐标

## 右键菜单

| 选项 | 功能 |
|------|------|
| **体型：小 (50px)** | 切换为小型角色 |
| **体型：中 (80px)** | 切换为中型角色（默认） |
| **体型：大 (110px)** | 切换为大型角色 |
| **穿透模式** | 勾选开关，`Ctrl+Shift+P` 也可切换 |
| **退出** | 关闭桌宠（`Ctrl+Q` 也可） |

## 体型切换

切换时有**缩放过渡动画**：
1. 设置目标缩放比例
2. 计算旧/新体型比作为起始缩放值
3. requestAnimationFrame 平滑过渡到 1.0
4. 同步通知主进程 `set-pet-size` IPC
5. 重新计算窗口布局（`Layout.applySize()`）

## 穿透模式

日常不遮挡操作的核心功能：

| 操作 | 方式 |
|------|------|
| 开启穿透 | 托盘菜单 / 右键菜单 / `Ctrl+Shift+P` |
| 关闭穿透 | 同上 |
| 启动时默认穿透 | `config.json` → `passthrough: true` |

穿透状态下鼠标事件（点击、移动）直达下层窗口，仅保留 `Ctrl+Shift+P` 全局快捷键。
