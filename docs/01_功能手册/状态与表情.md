# 状态与表情

## FSM 状态定义

由 `fsm/fsm-core.js` 引擎驱动，共 8 个状态：

| 状态 | 触发条件 | 动画描述 | 持续时间 |
|------|----------|----------|----------|
| **idle** | 默认 | 呼吸起伏 + 眨眼 + 眼球追踪 + 睡眠子状态 | 常驻 |
| **working** | 文件拖放 | 张嘴吃文件（nom 节奏 ~400ms）+ 碎屑粒子 + 手臂伸手 | 复制完成 |
| **happy** | 点击头部 / 双击 | 弹跳 + 双臂挥舞 + chirp 音效 + 心形粒子 | 1.5s / 2.5s |
| **surprised** | 点击身体 | 后退 + 双臂张开 + 抖动衰减 + boing 音效 + 火花粒子 | 1.5s |
| **alert** | 预留 | 蹲下蓄力 → 跳起 + 单臂指天 + 星星眼 + 星形粒子 | 常驻 |
| **bongo** | 输入"bongo" | 坐姿 + 双手交替拍击 + bongo 音效 + 音符粒子 | 10s |
| **listening** | 输入"听歌" / 音乐检测 | 戴耳机 + 闭眼摇摆 + 音符飘出 | 手动退出 / 音乐停止 |
| **sleeping** | idle 子状态（120s） | Zzz 气泡 + 闭眼 → 鼠标移动唤醒 + 伸懒腰 | 鼠标唤醒 |

## 状态生命周期

```
_transitionTo('happy'):
  → _exit('idle')    ← 离开 idle
  → _enter('happy')  ← 初始化 happy 状态（设置计时器等）
  → _update('happy') ← 每帧更新（检查是否 1.5s 已到，自动回 idle）
```

## 表情映射表

表情是**数据驱动**的，由眼型 + 嘴型 + 腮红三要素组合：

| 表情名 | 中文 | 眼型 (eyes) | 嘴型 (mouth) | 腮红 | 使用状态 |
|--------|------|-------------|-------------|------|----------|
| neutral | 平静 | round | none | ❌ | idle |
| happy | 开心 | arch | smile | ✅ | happy |
| working | 专注 | round | none | ❌ | working |
| open | 张嘴 | round | open | ❌ | working（进食瞬间） |
| surprised | 惊讶 | wide | open | ❌ | surprised |
| sleepy | 困倦 | half_closed | none | ❌ | idle（60s） |
| proud | 骄傲 | closed_arch | grin | ✅ | 预留 |
| alert | 提醒 | sparkle | none | ❌ | alert |
| sleeping | 睡觉 | closed | none | ❌ | idle（120s） |
| bongo | 敲击 | arch | grin | ✅ | bongo |
| listening | 听歌 | closed_arch | none | ✅ | listening |

## 眼型 (7 种)

| 眼型 | 像素绘制 | 说明 |
|------|----------|------|
| `round` | 2×2 方块 | 标准圆眼 |
| `arch` | 2×2 方块上移 1px | 开心弧形眼 |
| `wide` | 3×2 方块 | 惊讶大眼 |
| `closed` | 1px 高直线 | 闭眼 |
| `closed_arch` | 1px 高弧线 | 开心闭眼（听歌/骄傲） |
| `sparkle` | 2×2 方块 + 星形高光 | 星星眼（提醒） |
| `half_closed` | 2×1 方块（上半保留） | 困倦半闭眼 |

## 嘴型 (4 种)

| 嘴型 | 像素绘制 | 说明 |
|------|----------|------|
| `none` | 不绘制 | 无嘴 |
| `smile` | 上弧线 2px 宽 | 微笑 |
| `open` | 2×2 开口方块 | 张嘴 |
| `grin` | 3px 宽弧线 | 咧嘴笑 |
