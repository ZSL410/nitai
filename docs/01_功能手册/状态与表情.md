# 状态与表情

## FSM 状态定义

由 `behavior/state-machine.js` 引擎驱动，共 6 个状态（+ 1 个动态注册状态）：

| 状态 | 触发条件 | 动画描述 | 持续时间 |
|------|----------|----------|----------|
| **idle** | 默认 | 呼吸起伏 + 眨眼 + 眼球追踪 + 微晃动 + 偶发微动作 | 常驻 |
| **walking** | AI 闲逛 / 召唤 / 好奇 | 行走步态动画（手脚交替）+ 窗口跟随 | 到达目标 |
| **curious** | 走到目标后 | 歪头 + 微张嘴 + 好奇表情 + 泡泡 | 2.5s |
| **happy** | 点击头部 / 双击 / 对话 | 弹跳 + 举手 + chirp 音效 + 心形粒子 + 泡泡 | 1.5-2.5s |
| **surprised** | 点击身体 | 跳起 + 张大嘴 + boing 音效 + 火花粒子 + 泡泡 | 2s |
| **sleepy** | 60s 无交互 | 半闭眼 + 缩小 + 偶发打哈欠 | 鼠标唤醒 |
| **working** | 文件拖放（动态注册） | 张嘴吃文件 + 碎屑粒子 + 进度条 | 复制完成 |

## 状态生命周期

```
_transitionTo('happy'):
  → _exit('idle')    ← 离开 idle
  → _enter('happy')  ← 初始化 happy（弹跳 + 音效 + 粒子）
  → _update('happy') ← 每帧更新（检查是否超时，自动回 idle）
```

## 表情映射表

表情是**数据驱动**的，由眼型 + 嘴型 + 腮红三要素组合：

| 表情名 | 中文 | 眼型 (eyes) | 嘴型 (mouth) | 腮红 | 使用状态 |
|--------|------|-------------|-------------|------|----------|
| neutral | 平静 | round | none | ❌ | idle / walking |
| happy | 开心 | arch | smile | ✅ | happy |
| surprised | 惊讶 | wide | open | ❌ | surprised |
| curious | 好奇 | wide | smile | ✅ | curious |
| sleepy | 困倦 | half_closed | none | ❌ | sleepy |
| dizzy | 眩晕 | closed_arch | open | ❌ | 预留 |
| excited | 兴奋 | sparkle | grin | ✅ | 预留 |
| alert | 警惕 | wide | none | ❌ | 预留 |
| grin | 咧嘴笑 | arch | grin | ✅ | 预留 |
| content | 满足 | closed | smile | ✅ | 预留 |
| love | 喜爱 | heart | smile | ✅ | 预留 |
| shock | 震惊 | wide | o | ❌ | 预留 |

## 眼型 (8 种)

| 眼型 | 像素绘制 | 说明 |
|------|----------|------|
| `round` | 2×2 方块 + 1px 高光 | 标准圆眼 |
| `arch` | 2×1 方块（上排） | 开心弧形眼（^ ^） |
| `wide` | 3×3 方块 + 1px 高光 | 惊讶大眼 |
| `closed` | 3×1 直线 | 闭眼（— —） |
| `closed_arch` | 3×1 弧线 | 开心闭眼（> <） |
| `half_closed` | 2×2 方块（保留下半） | 困倦半闭眼（= =） |
| `sparkle` | 十字星形高光 | 星星眼（✦ ✦） |
| `heart` | 心形像素图案 | 爱心眼（♥ ♥） |

## 嘴型 (5 种)

| 嘴型 | 像素绘制 | 说明 |
|------|----------|------|
| `none` | 不绘制 | 无嘴 |
| `smile` | 2px 宽上弧线 | 微笑 |
| `open` | 2-3px 高开口 | 张嘴（可调深度 0-1） |
| `grin` | 4px 宽 + 牙齿 | 咧嘴大笑 |
| `o` | 圈形开口 | 惊讶 O 嘴 |
