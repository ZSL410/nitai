# 技术栈

## 运行时

| 组件 | 版本 | 用途 |
|------|------|------|
| Electron | ^41.7.1 | 桌面应用框架，提供透明窗口 + 系统托盘 + IPC |
| Node.js | (内置) | 文件系统操作、HTTP 请求、进程管理 |
| Chromium | (内置) | Canvas 2D 渲染、Web Audio API |

## 渲染技术

| 技术 | 用途 |
|------|------|
| Canvas 2D | 像素角色绘制（`imageSmoothingEnabled = false` 保证像素无抗锯齿） |
| CSS | 透明背景、气泡样式、箭头伪元素（`::after`） |
| requestAnimationFrame | 60fps 动画循环（低功耗模式降至 15fps） |

## 音频

| 技术 | 用途 |
|------|------|
| Web Audio API | 音效合成（chirp/yawn/boing/nom/bongo），懒初始化 AudioContext |
| 无外部音频文件 | 所有音效由 JavaScript 合成 |

## 主进程模块

| 模块 | 关键技术 |
|------|----------|
| `window.js` | `BrowserWindow` transparent + alwaysOnTop + `setIgnoreMouseEvents` |
| `tray.js` | `Tray` + `Menu` 系统托盘 |
| `ipc-handlers.js` | `ipcMain.on` 扁平参数传递 |
| `music-detector.js` | `child_process.exec('tasklist')` 进程检测 |
| `position-store.js` | `fs.writeFileSync` JSON 持久化 |

## 开发环境

| 工具 | 用途 |
|------|------|
| Windows 10/11 | 目标运行平台（原生 Electron） |
| WSL2 (Ubuntu) | 代码编辑 + Git 操作（不能运行 GUI） |
| Node.js | npm 包管理 |
| Git | 版本控制 |

## 版本要求

| 依赖 | 最低版本 |
|------|----------|
| Node.js | ≥ 18 |
| npm | ≥ 9 |
| Windows | 10+ (64-bit) |
| 磁盘空间 | ~200MB（含 Electron 和 node_modules） |
