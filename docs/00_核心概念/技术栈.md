# 技术栈

## 运行时

| 组件 | 版本 | 用途 |
|------|------|------|
| Electron | ^41.7.1 | 桌面应用框架，提供透明窗口 + 系统托盘 + IPC |
| Node.js | (内置) | 文件系统操作、HTTP 请求、进程管理 |
| Chromium | (内置) | Canvas 2D 渲染、Web Audio API |

## 渲染技术

| 技术 | 用途 |
|------|------|
| Canvas 2D | 像素角色绘制（`imageSmoothingEnabled = false` 保证像素无抗锯齿） |
| CSS | 透明背景、气泡样式、箭头伪元素（`::after`） |
| requestAnimationFrame | 60fps 游戏循环（帧率无关阻尼算法保证一致体验） |

## 音频

| 技术 | 用途 |
|------|------|
| Web Audio API | 音效合成（chirp/boing/nom/footstep），懒初始化 AudioContext |
| 无外部音频文件 | 所有音效由 JavaScript 合成 |

## 主进程模块

| 模块 | 关键技术 |
|------|----------|
| `window.js` | `BrowserWindow` transparent + alwaysOnTop + `setIgnoreMouseEvents` + 全屏召唤覆盖层 |
| `tray.js` | `Tray` + `Menu` 系统托盘 |
| `ipc-handlers.js` | `ipcMain.on` 扁平参数传递，含 move-window/start-summon/summon-target |
| `music-detector.js` | `child_process.exec('tasklist')` 进程检测 |
| `position-store.js` | `fs.writeFileSync` JSON 持久化 |

## 渲染进程模块

| 模块 | 关键技术 |
|------|----------|
| `character/pixel-renderer.js` | Canvas 2D 16×20 网格绘制 + 肢体动画 + 表情渲染 |
| `character/animations.js` | 生命感系统（呼吸正弦波 + 随机眨眼 + 晃动） |
| `behavior/state-machine.js` | FSM 6 状态（idle/walking/curious/happy/surprised/sleepy） |
| `behavior/ai-controller.js` | 层次化决策（闲逛/好奇心/睡眠/微动作） |
| `movement/walk-controller.js` | 世界坐标行走 + `win.setPosition()` 窗口同步 |
| `movement/easing.js` | damp()/lerp()/Penner 缓动函数 |

## 开发环境

| 工具 | 用途 |
|------|------|
| Windows 10/11 | 目标运行平台（原生 Electron） |
| WSL2 (Ubuntu) | 代码编辑 + Git 操作（不能运行 GUI） |
| Node.js | npm 包管理 |
| Git | 版本控制 |

## 版本要求

| 依赖 | 最低版本 |
|------|----------|
| Node.js | ≥ 18 |
| npm | ≥ 9 |
| Windows | 10+ (64-bit) |
| 磁盘空间 | ~200MB（含 Electron 和 node_modules） |
