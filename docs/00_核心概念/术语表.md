# 术语表

> 拟态项目专用术语定义。保持统一命名，避免沟通歧义。

| 术语 | 英文 | 定义 |
|------|------|------|
| **拟态** | nitai / Mimic | 项目名称。取自动物学 mimicry（拟态），意为融入桌面环境的小生物。也是全局命名空间 `window.Mimic`。 |
| **桌宠** | desktop pet | 运行在桌面上的虚拟宠物角色，16×20 像素网格小人。 |
| **晓风** | xiaofeng | 晓风智能体主项目，拟态的后端。运行在 `http://127.0.0.1:5001`。 |
| **FSM** | Finite State Machine | 有限状态机。驱动角色动画和行为的核心引擎（`fsm/fsm-core.js`）。 |
| **气泡** | bubble / speech bubble | 角色上方的对话框。消息队列系统，从嘴巴上方"长"出，带箭头尾巴。 |
| **粒子** | particle | 飘散的特效元素。6 种：心形(❤)、星形(✦)、火花(·)、碎屑、音符(♪)、雨滴(💧)。 |
| **表情** | expression | 眼型 + 嘴型 + 腮红的组合。数据驱动，11 种预定义组合。 |
| **眼型** | eye type | 眼睛的像素绘制模式。7 种：round / arch / wide / closed / closed_arch / sparkle / half_closed。 |
| **嘴型** | mouth type | 嘴巴的像素绘制模式。4 种：none / smile / open / grin。 |
| **穿透模式** | passthrough | 鼠标事件穿透到下层窗口（`setIgnoreMouseEvents`）。`Ctrl+Shift+P` 切换。 |
| **低功耗模式** | low-power mode | idle 30s 后帧率降至 15fps。鼠标移动立即恢复 60fps。 |
| **布局公式** | layout formula | `winH = petSize + 2×max(overhead, BOTTOM_MARGIN)` 窗口高度计算。保证角色居中 + 气泡可见。 |
| **IPC** | Inter-Process Communication | Electron 主进程与渲染进程通信。拟态使用扁平参数传递。 |
| **Web Audio** | Web Audio API | 浏览器内置音频合成 API。拟态用它合成 5 种音效，无需外部文件。 |
| **子状态** | sub-state | idle 状态内部的状态切换（sleepy→sleeping），不触发 FSM 全局转换。 |
| **消息队列** | message queue | 气泡系统的多条消息顺序显示机制。每条间隔 2 秒。 |
| **音乐检测** | music detection | 后台 3s 轮询系统进程列表，检测 Spotify/QQ音乐/网易云等播放器。 |
| **Pointer Capture** | Pointer Capture | 指针锁定机制，用于可靠的窗口拖拽。WSL2 下退化为 document 级追踪。 |
| **坐标硬编码** | hardcoded coordinates | 已知问题：部分绘制逻辑使用固定像素偏移而非基于 petCX/petCY 动态计算。 |
