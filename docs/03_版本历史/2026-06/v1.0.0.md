# v1.0.0 — 拟态独立后首个正式版本

> 日期：2026-07-02 | 版本：v1.0.0 | 架构重构 + 文件整理

## 背景

拟态于 2026-06-28 从晓风主项目分离为独立 Git 仓库（`D:\nitai\`）。v1.0.0 是独立后首个正式版本，包含一次完整的架构重构——从"静态窗口包裹角色"转变为"角色拥有世界坐标，在整个桌面上自由移动"。

## 核心变更

### 🏗️ 范式转换：世界坐标系统

**v1.0 前**：角色禁锢在静态窗口中，窗口位置固定。移动只能通过拖拽。

**v1.0.0**：角色拥有屏幕世界坐标（`worldX`, `worldY`），窗口跟随角色移动。整个电脑桌面都是它的领地。

- 新增 `mimic.js` 世界坐标状态（`worldX`/`worldY`）
- 新增 `movement/walk-controller.js`：帧率无关阻尼缓动，80px/s 慢走 / 160px/s 跑步
- 新增 `movement/easing.js`：`damp()`/`lerp()`/Penner 缓动函数库
- 窗口通过 `win.setPosition()` 每帧同步到角色位置
- 屏幕边界自动限制（`moveWindowTo()` 内置 clamp）

### 🔮 召唤系统

三通道触发角色移动到桌面任意位置：

| 触发方式 | 操作 |
|----------|------|
| 右键菜单 | → "召唤到这里" → 点击桌面任意位置 |
| 三击角色 | 连击三次角色 → 自动进入召唤模式 |
| 文字指令 | 按 Enter → 输入 "过来" / "召唤" / "summon" |

实现原理：主进程创建临时全屏透明 BrowserWindow 捕获一次点击 → 获取屏幕坐标 → 角色通过 `walkTo()` 走过去（非瞬移）。

### 🧠 层次化自主 AI

新增 `behavior/ai-controller.js`，每 3 秒决策一次：

- **35%** 概率：随机闲逛到屏幕某处
- **15%** 概率：微动作（伸懒腰/歪头/小跳）
- **50%** 概率：继续当前 idle 状态
- **好奇心**：鼠标靠近（150-500px）时偶尔走过去查看
- **睡眠**：60 秒不活动 → 进入 sleepy 状态（半闭眼 + 打哈欠）
- **Ctrl+W**：切换自主闲逛开关

### 💓 生命感系统

始终运行的基础动画，营造"活着的"感觉：

- **呼吸**：3 秒周期正弦波缩放振荡（振幅 ±2%）
- **眨眼**：随机间隔 2-6 秒，每次持续 100ms
- **微晃动**：5 秒周期水平偏移（振幅 ±0.3 格）
- **偶发动作**：伸懒腰（举臂）、歪头、小跳

### 🚶 移动系统

- **帧率无关**：`damp(a, b, λ, dt)` 指数衰减算法
- **行走动画**：4 帧步态循环（手脚交替摆动）
- **速度分级**：慢走 80px/s，跑步 160px/s
- **方向感知**：根据水平位移判断朝向

### 🗂️ 文件结构整理

- 代码目录（`拟态开发版/`）与文档目录（`docs/`）完全分离
- 代码文件按功能分目录：`character/` `behavior/` `movement/` `interaction/` `effects/` `audio/`
- 删除旧模块：`fsm/`（被 `behavior/` 替代）、`rendering/`（被 `character/` 替代）、`layout.js`（被世界坐标系统替代）
- `main.js` 移入 `main/`，`particles.js` 移入 `effects/`，`audio.js` 移入 `audio/`

### 🎨 角色视觉

- 16×20 网格像素小人，完整肢体动画
- 12 种表情：neutral/happy/surprised/curious/sleepy/dizzy/excited/alert/grin/content/love/shock
- 7 种眼型 + 5 种嘴型 + 腮红 + 耳机配件
- 三种体型（50/80/110px），缩放过渡动画

## 已知问题

| 问题 | 严重程度 | 说明 |
|------|----------|------|
| 左上角卡死 | 🟡 中 | 角色偶尔会走到屏幕左上角卡住，与边界检测和路径规划有关 |
| 拖拽后位置不准确 | 🟡 中 | 手动拖拽后世界坐标与窗口实际位置可能不同步 |
| 气泡遮挡 | 🟢 低 | 气泡偶尔遮挡角色面部，与窗口高度自适应有关 |
| 窗口闪烁 | 🟢 低 | 快速行走时窗口位置更新可能造成轻微视觉抖动 |
