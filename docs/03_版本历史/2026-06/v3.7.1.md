# v3.7.1 — Phase 1 交互体验优化

> 日期：2026-06-27 | 版本：v3.7.1 | 晓风期内最后版本

## 变更内容

### 🎯 眼球跟随鼠标增强

文件：`renderer/interaction/eye-tracking.js` — 重写

- 基于**角色头部中心**（而非窗口中心）计算鼠标偏移量
- 眼珠移动范围扩大至 ±2 格水平、±1 格垂直（严格限制在头部区域内）
- 鼠标离开窗口时眼珠平滑回到中心
- 配合 `pixel-character.js` 支持浮点偏移渲染

### 🖱️ 点击部位反馈升级

文件：`renderer/interaction/click.js` — 增强

| 操作 | 触发状态 | 音效 | 粒子 | 持续 |
|------|----------|------|------|------|
| 点击头部 | happy | chirp | 心形 ❤ | 1.5s |
| 点击身体 | surprised | boing | 火花 · | 1.5s |
| 双击 | happy（旋转弹跳） | boing | 火花 · | 2.5s |

### 🔋 低功耗空闲模式

文件：`renderer/app.js` — 新增帧率控制

- 角色处于 `idle` 且连续 30 秒无鼠标移动 → 降至 15fps
- 帧跳过机制：每 4 帧绘制 1 帧（60÷15=4）
- 鼠标移动立即恢复 60fps 全速绘制
- 切换体型时自动重置空闲计时器

## 技术细节

```javascript
// 低功耗帧跳过逻辑
const idleTime = now - M.lastActivity;
const isIdle = M.FSM.state === 'idle';

if (isIdle && idleTime >= 30000) {
  M._idleFrameCounter++;
  if (M._idleFrameCounter >= 4) {
    M._idleFrameCounter = 0;
    draw();  // 每 4 帧绘制 1 次
  }
} else {
  draw();  // 每帧绘制
}
```

## 已知问题

### 坐标硬编码残留

部分绘制逻辑中仍存在硬编码像素偏移。具体表现为：
- 角色视觉位置偏上（尤其在非 80px 体型）
- 气泡箭头遮挡/偏差

影响版本 v2.0.0 → v3.7.1，计划 v3.8.0 或独立后 v1.0.0 修复。

### 眼球追踪抖动

一阶 lerp（0.28）在快速移动时偶有视觉抖动。影响轻微。
