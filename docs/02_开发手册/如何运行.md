# 如何运行

## Windows 原生（推荐）

### 首次安装

```cmd
cd D:\nitai\拟态开发版
npm install
```

### 启动

```cmd
cd D:\nitai\拟态开发版 && npm start
```

桌宠将以透明窗口出现在屏幕右下角（或上次关闭时的位置）。

### 关闭

三种方式：
- 右键菜单 → 退出
- 系统托盘菜单 → 退出
- `Ctrl+Q`

## WSL2 开发环境

WSL2 **不能**正常运行 GUI 应用。桌宠必须在 Windows 原生环境启动。

WSL2 仅用于：
- 代码编辑
- Git 操作
- 运行 `git status` / `git diff` 等

### 紧急调试（不稳定）

```bash
cd /mnt/d/nitai/拟态开发版 && npm run start:wsl
```

此命令使用 `--no-sandbox --disable-gpu --no-zygote` 参数，在 WSL2 中勉强运行。可能遇到：
- 渲染异常
- 窗口位置错误
- 系统托盘不可用

**不推荐**日常使用此方式。

## 命令行参数

| 脚本 | 用途 |
|------|------|
| `npm start` | Windows 原生启动 |
| `npm run start:wsl` | WSL2 紧急调试 |

`package.json` 中定义：

```json
{
  "scripts": {
    "start": "electron .",
    "start:wsl": "LD_LIBRARY_PATH=./electron-libs electron . --no-sandbox --disable-gpu --no-zygote"
  }
}
```

## 配置文件

首次启动后，可编辑 `拟态开发版/config.json` 自定义行为：

```json
{
  "targetDir": "../02_数据输入",
  "backendUrl": "http://127.0.0.1:5001",
  "apiEnabled": false,
  "defaultSize": 80,
  "passthrough": false,
  "musicDetect": true,
  "musicCheckInterval": 3,
  "musicPlayers": ["spotify", "qqmusic", "cloudmusic", "netease", "kwmusic", "kugou"]
}
```

## 与晓风后端一起运行

1. 先启动晓风后端（确保端口 5001 可用）
2. 再启动拟态桌宠
3. 在 `config.json` 中设置 `"apiEnabled": true`
