# 如何调试

## 打开 DevTools

### 方式一：代码触发

在渲染进程代码中调用：

```javascript
const { remote } = require('electron');
remote.getCurrentWindow().webContents.openDevTools({ mode: 'detach' });
```

### 方式二：主进程启动参数

临时在 `main/main.js` 中添加：

```javascript
win.webContents.openDevTools({ mode: 'detach' });
```

### 方式三：快捷键（未实现）

当前版本没有内置 DevTools 快捷键。可自行在 `mimic.js` 的 keydown 事件中添加。

## 调试 IPC 通信

在 DevTools Console 中手动发送 IPC：

```javascript
// 测试窗口移动
window.Mimic.ipc.send('move-window', 500, 300);

// 测试穿透切换
window.Mimic.ipc.send('toggle-passthrough');

// 查看当前状态
window.Mimic.FSM.state;
window.Mimic.FSM.getExpression();
```

## 调试 FSM 状态

```javascript
// 手动切换状态
window.Mimic.FSM.transitionTo('happy');
window.Mimic.FSM.transitionTo('curious');
window.Mimic.FSM.transitionTo('sleepy');

// 查看当前所有状态
console.log({
  state: M.FSM.state,
  prevState: M.FSM.prevState,
  stateTimer: M.FSM.stateTimer,
  expression: M.FSM.getExpression()
});
```

## 调试行走

```javascript
const M = window.Mimic;

// 查看世界坐标
console.log({ worldX: M.worldX, worldY: M.worldY });

// 查看行走状态
console.log({
  isWalking: M.isWalking,
  hasTarget: M.Walk.hasTarget,
  targetX: M.Walk.targetX,
  targetY: M.Walk.targetY,
  distance: M.Walk.distanceToTarget()
});

// 手动走到指定位置
M.Walk.walkTo(800, 400, 'run', () => console.log('arrived!'));
```

## 调试角色渲染

```javascript
const M = window.Mimic;

// 查看渲染参数
const B = M.Rendering.getBounds();
console.log({
  petSize: B.petSize,
  cellSize: B.cellSize,
  gx: B.gx, gy: B.gy,
  headTop: B.headTop,
  mouthX: B.mouthX, mouthY: B.mouthY,
});

// 查看动画状态
console.log({
  bobOffset: M.Anim.bobOffset,
  bodyScale: M.Anim.bodyScale,
  eyePupilX: M.Anim.eyePupilX,
  walkPhase: M.Anim.walkPhase,
});
```

## 调试粒子

```javascript
const M = window.Mimic;
if (M.Particles) {
  M.Particles.burst('heart', 5, { x: 8, y: 3 });
  M.Particles.burst('note', 3, { x: 10, y: 2 });
}
```

## 调试 AI

```javascript
const M = window.Mimic;

// 查看 AI 状态
console.log({
  wanderEnabled: M.wanderEnabled,
  nextDecision: M.AI._nextDecisionTime,
});

// 手动触发闲逛
M.AI.forceWander();

// 切换闲逛开关
M.AI.toggleWander();
```

## 常见调试场景

| 问题 | 调试方法 |
|------|----------|
| 角色不走 | 检查 `M.wanderEnabled` 是否为 true；检查 `M.Walk.hasTarget` |
| 召唤无反应 | 检查 IPC 通道：DevTools → Network 或 Console 查看 summon-target 事件 |
| 眼球不跟随 | 检查 `M.FSM.state === 'idle'` 是否为 true |
| 音效不播放 | 检查 AudioContext 状态（可能被浏览器策略阻塞） |
| 文件拖放无反应 | 检查 `targetDir` 路径是否存在 |
| 角色卡在左上角 | `M.worldX = 800; M.worldY = 400;` 手动重置位置 |

## 性能分析

在 DevTools Performance 面板录制动画循环：

1. 打开 DevTools → Performance 标签
2. 点击录制
3. 操作桌宠（拖拽/点击/切换体型/行走）
4. 停止录制 → 查看帧率和调用栈
