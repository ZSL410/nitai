# 如何添加新表情

## 概述

表情系统是数据驱动的。添加新表情只需修改表情定义文件，如涉及新眼型/嘴型则需额外修改渲染层。

## 步骤一：定义表情（expressions.js）

编辑 `renderer/character/expressions.js`，在 `EXPRESSIONS` 对象中添加新条目：

```javascript
const EXPRESSIONS = {
  // ... 已有表情 ...

  excited: {
    eyes: 'sparkle',      // 使用已有眼型
    mouth: 'grin',        // 使用已有嘴型
    cheek: true,          // 显示腮红
  },
};
```

## 步骤二：在状态中使用

在 `renderer/behavior/state-machine.js` 的 FSM handler 中引用：

```javascript
// 静态表情直接返回字符串
FSM._handlers.happy = {
  expression: 'excited',  // 新表情
  // ...
};

// 或动态返回
expression: function() {
  if (this.context.intensity === 'high') return 'excited';
  return 'happy';
},
```

## 步骤三（可选）：添加新眼型

如果现有 8 种眼型不够用，在 `renderer/character/pixel-renderer.js` 的 `drawFace()` 函数中添加新 case。

### 眼型绘制规范

眼型在 `drawFace()` 的 switch 语句中定义，每个 case 使用 `drawCell()` 辅助函数。参考现有 `heart` 眼型的实现：

```javascript
case 'heart':
  [lx, rx].forEach(hx => {
    drawCell(ctx, hx,   ey,   C.eye, S, gx, gy);
    drawCell(ctx, hx+1, ey,   C.eye, S, gx, gy);
    drawCell(ctx, hx-1, ey+1, C.eye, S, gx, gy);
    drawCell(ctx, hx,   ey+1, C.eyeSpark, S, gx, gy);  // 高光
    // ...
  });
  break;
```

### 关键约束

- 眼型位置基于 `lx`/`rx`（已含瞳孔偏移 px/py）和 `ey` 计算
- 使用 `C.eye`（瞳孔色）、`C.eyeWhite`（高光）、`C.eyeSpark`（金色高光）
- 所有坐标基于网格单位（`S` = cellSize）

## 步骤四（可选）：添加新嘴型

在 `drawFace()` 的 mouth switch 语句中添加新 case。参考现有 `o` 嘴型：

```javascript
case 'o':
  e(6, 4); e(7, 4); e(8, 4); e(9, 4);
  e(6, 5); e(9, 5);
  e(6, 6); e(9, 6);
  e(7, 6); e(8, 6);
  break;
```

## 步骤五：无需修改 index.html

表情定义在 `character/expressions.js` 中，已在 `index.html` 加载链中，无需新增 `<script>` 标签。

## 验证

1. 启动桌宠
2. 打开 DevTools Console
3. 手动触发新表情验证渲染：

```javascript
// 临时修改 FSM 返回新表情
const orig = M.FSM.getExpression;
M.FSM.getExpression = () => M.EXPRESSIONS.excited;
M.Rendering.draw();

// 恢复
M.FSM.getExpression = orig;
```

4. 在三种体型（50/80/110）下验证表情居中、比例正确
