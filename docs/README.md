## 快速命令

```cmd
# Windows 原生启动
cd D:\nitai\拟态开发版 && npm start

# 安装依赖（首次）
cd D:\nitai\拟态开发版 && npm install

# WSL 紧急调试（不稳定）
cd /mnt/d/nitai/拟态开发版 && npm run start:wsl
```
