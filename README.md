# 拟态（nitai）

> 晓风智能体的桌面像素桌宠 — 一只住在你屏幕右下角的小家伙。

## 项目定位

拟态是 [晓风 Agent](https://github.com/ZSL410/xiaofeng) 体系的 Windows 桌面宠物，以 16×20 像素网格小人呈现。它是晓风的**数据入口**（文件拖放）、**情绪出口**（表情反馈）和**陪伴界面**（交互反应）。

## 技术栈

| 层面 | 技术 |
|------|------|
| 运行时 | Electron 41 |
| 渲染 | Canvas 2D（像素无抗锯齿） |
| 状态管理 | FSM 有限状态机（8 状态） |
| 音效 | Web Audio API 合成 |
| 通信 | HTTP API → 晓风后端 (`127.0.0.1:5001`) |

## 快速启动

```cmd
cd D:\nitai\拟态开发版 && npm start
```

首次运行先安装依赖：`cd D:\nitai\拟态开发版 && npm install`

## 核心功能

透明窗口 · 文件拖拽 · FSM 状态机 · 表情系统（11 种）× 眼球追踪 · 点击反应 · 气泡反馈 · 听歌模式 · 穿透模式（Ctrl+Shift+P）· 低功耗模式（30s → 15fps）· 粒子特效（6 种）· 音效合成（5 种）· 位置记忆

## 与晓风通信

| 接口 | 方法 | 用途 |
|------|------|------|
| `/api/file/process` | POST | 文件复制完成后通知后端 |
| `/api/chat` | POST | 对话消息（可选，无后端时 Mock） |

通信开关由 `拟态开发版/config.json` 中 `apiEnabled` 控制。
